# Assignment 3 Reflection

## Manual and AI-Generated Commit Messages

My manual commit messages were usually shorter and focused on the change I had just completed. The AI-generated messages provided more detail about the purpose of each commit. If I returned to this project one year later, I would prefer the AI-style messages because they make the project history easier to understand.

## AI Code Review

The AI code review identified a real issue with numeric input validation. The original scripts removed missing values but did not clearly handle text or incorrectly formatted values. I updated the Python and R scripts so that the selected columns are converted to numeric data and the analysis stops when there are not enough valid observations. The review did not identify any major problem with the regression calculations themselves.

## Manual and AI-Generated READMEs

My manual README was shorter and focused on the basic purpose of the repository. The AI-generated README included more detailed file descriptions, setup instructions, commands, and expected outputs. The AI version would probably be more useful to a new contributor because it explains how to reproduce the analysis step by step.

## Future Workflow Habit

One habit I would continue using is checking and validating input data before fitting a model. I would also use clearer commit messages that explain both what changed and why the change was made.
