# Linear Regression in Python and R

This repository contains my work for Assignment 2, which focuses on building and comparing simple linear regression models in Python and R. The analysis uses `regression_data-1.csv`, which contains the variables `YearsExperience` and `Salary`. The goal is to examine how salary changes with years of experience, visualize the relationship with a scatter plot, fit a linear regression model, overlay the regression line, and evaluate the model using R-squared.

The `manual` folder contains the version I created by following the assignment instructions. It includes Python and R Jupyter Notebooks, exported HTML files, command-line scripts, and the regression plot images generated by the scripts. The Python analysis uses pandas, matplotlib, and scikit-learn, while the R analysis uses R, `lm()`, and ggplot2. Both models show a positive linear relationship between years of experience and salary.

The `ai` folder contains a separate AI-assisted version of the same assignment. AI was used to regenerate both notebooks and both command-line scripts. I tested the generated code, corrected errors, and confirmed that the scripts produced the required output images. The most important prompts used during this process are documented in `ai/PROMPTS.md`.
